# Agents Roadmap

Агентский воркфлоу для соло-разработки: AI-агенты закрывают роли членов команды, которых нет. Разработчик выступает в роли Tech Lead — принимает финальные решения.

## Команда агентов

| Агент | Роль | Вопрос, на который отвечает |
|---|---|---|
| PR Review Agent | Проверяет Pull Request перед мержем: Conventional Commits, соответствие OpenAPI-контракту, архитектурные соглашения. | "Это готово к мержу?" |
| Architect Agent | Следит за архитектурной целостностью. Фиксирует решения в `DECISIONS.md`. Проверяет новые фичи на соответствие принятым решениям. | "Это правильно спроектировано?" |
| QA Agent | Пишет тесты на реализованный код, валидирует что фича работает как заявлено. | "Это работает?" |

## Порядок внедрения

- [x] PR Review Agent
- [ ] Architect Agent
- [ ] QA Agent

## Опциональные роли

UI Tester, Performance Monitor, Security Auditor — появляются при конкретной необходимости, не нужны на старте.
